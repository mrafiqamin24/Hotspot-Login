# Home.Net GitHub Pages Prototype

Folder ini adalah versi prototipe statis dari template Home.Net untuk di-upload ke GitHub Pages.

## Cara pakai

1. Upload semua isi folder ini ke repository GitHub.
2. Di GitHub, buka Settings > Pages.
3. Pilih branch yang berisi file ini, lalu set folder ke `/root`.
4. Buka URL GitHub Pages yang diberikan GitHub.

## Catatan

- `index.html` adalah halaman login demo.
- Placeholder MikroTik seperti `$(username)` sudah diganti data contoh agar tampil rapi di browser biasa.
- File asli project di luar folder ini tidak diubah.
